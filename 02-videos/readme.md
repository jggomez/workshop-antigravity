# Videos

This directory contains the audiovisual resource library for the workshop. These videos demonstrate the practical implementation of **Spec-Driven Development (SDD)** and **Vibe Engineering**, moving from initial conceptualization to advanced utilization of **Google Antigravity**.

---

## Series: Mastery in the Agentic AI-First Era

### 1. Vibe Coding: The End of Programming or a New Era of Superpowers?
* **Description**: A critical introduction to the paradigm shift. This video explores how the developer transitions from a manual coder to an orchestrator guiding the AI.
* **Key Topics**:
    * **Fundamentals**: The relationship between the developer and the LLM.
    * **Business Vision**: Accelerating from idea to MVP in an afternoon.
    * **Architecture**: The necessity of a technical blueprint before execution.
    * **Google Antigravity**: Introduction to the industrial power of an AI-native IDE.
* **🔗 Watch Video**: [https://youtu.be/iHMeoenQAL0](https://youtu.be/iHMeoenQAL0?si=62iB9Nh3ZEpoRaCl)

### 2. Vibe Coding Paradox Part 2: Give the Business a Brain First
* **Description**: Focused on avoiding the "Vibe Coding Hangover". Learn that before requesting code, you must provide the AI with robust business context.
* **Key Topics**:
    * **Needs Definition**: Extracting real value to prevent AI "hallucinations".
    * **Pro User Stories**: Creating technical acceptance criteria that the AI can execute deterministically.
    * **Smart Vision Board**: A practical case study on setting up initial project context.
* **🔗 Watch Video**: [https://youtu.be/hatexcSLXX8](https://youtu.be/hatexcSLXX8?si=QkslmKYekgzWeN6U)

### 3. Prototyping in Minutes: Using AI to Design Our App
* **Description**: Technical demonstration of speed within the "Vibe Zone". Ideal for initial discovery and visual validation phases.
* **Key Topics**:
    * Stitch interface and intelligent layouts.
    * UI generation based on structured prompts.
    * Setting the stage for the SDD development phase.
* **🔗 Watch Video**: [https://youtu.be/gHPRiFudilo](https://youtu.be/gHPRiFudilo?si=2GFDwAWGFnRSmZb3)

### 4. Refining the Architectural Blueprint with AI: Patterns and SDD
* **Description**: The transition toward rigorous engineering. This video covers using AI to make architecture decisions based on data and logic rather than just intuition.
* **Key Topics**:
    * **Quality Attributes (NFRs)**: Defining precise metrics such as security and performance.
    * **Architectural Trade-offs**: Using AI to justify design patterns (Hexagonal, Event-Driven).
    * **Technical Specification**: Documenting the "why" behind every component.
* **🔗 Watch Video**: [https://youtu.be/HU-YOTcjD5k](https://youtu.be/HU-YOTcjD5k?si=1kbgBwyba_lOIYmz)

### 5. Google Antigravity: Setup & Deep Dive
* **Description**: The definitive tutorial on the native IDE for the agentic era. It explores agent autonomy within the engineer's workflow.
* **Key Topics**:
    * **Setup & Governance**: Defining *Rules* and *Guardrails* to maintain control over the agent.
    * **MCP Ecosystem**: Connecting agents to real-time data and external documentation.
    * **Agent Manager**: Managing artifacts, the knowledge base, and the testing playground.
* **🔗 Watch Video**: [https://youtu.be/Icm48g0OH3Y](https://youtu.be/Icm48g0OH3Y?si=LeN2gqgU9I8vnW1O)
